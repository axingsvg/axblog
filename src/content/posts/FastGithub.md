# 软件介绍

FastGithub 是一个开源的跨平台 GitHub 加速神器，通过安全的 DNS 服务请求 GitHub 的 IP 地址，选择最优的 IP 进行反向代理，从而有效解决网络限制访问失败和速度缓慢的问题。
